Perception

Template Tracker implemented using Lucas Kanade tracking algorithm

In this project we tracked objects in different video sequences by implementing the Lucas kanade tracking Algorithm. We obtain an optical flow of the template which helps us to keep the object of interest in track. We also implemented a weighted version of the Least square equation, essentially giving more weight to track the center pixels for the object template and reducing the effect of outlier pixels. (PS: Yes, its not perfect, will have to fiddle around with the weights!)
